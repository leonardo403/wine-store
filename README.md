# wine-store
 
 O computador ou servidor deve ter configurado um database Mysql. Depois devemos executar os
 comandos a seguir.
 ´´
 npm install
 yarn install
 ´´
 ## Backend
 Para executar o backend:
 ´´
 cd backend
 php -S localhost:7400 -t public
 ´´
## Api
localhost:7400/produto/api

## Frontend
 Para executar o Frontend:
 ´´
 cd Frontend
 yarn start
 ´´
